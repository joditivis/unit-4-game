Crystal Collector

How to Play:

You will be given a random number at the start of the game located under Number to Match.

Each of the crystals below have a set number value that is hidden until you click on it. By clicking one of the four, you will add that crystals value to your guessed number.

You win by trying to make your guessed number equal to the Number to Match.

Each time you play, the number value of each crystal will change.
